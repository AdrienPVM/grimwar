# Plan 13.8 — Sous-choix d'ascendance niveau 1 (Drakéide / Tieffelin / Elfe / Gnome / Goliath / Humain)

> **Plan B du jalon 1 « complétude SRD 5.2.1 niveau 1 ».** Consomme 13.7 (schema v2 + `public/data/ancestries.json` enrichi). 13.9 et 13.10 peuvent être livrés en parallèle de 13.8 une fois 13.7 mergé ; on garde la séquentialité par défaut pour la lisibilité des reviews.

## Goal

Faire en sorte que le wizard de création de personnage **impose** au joueur tous les sous-choix d'ascendance que le SRD 5.2.1 demande au niveau 1, avec un **pattern pédagogique débutant** (panneau d'aide par sous-choix, vocabulaire FR sans jargon, exemples concrets), et faire en sorte que la fiche **affiche correctement** les effets des sous-choix dans les modes existants (Combat pour le souffle Drakéide, Magie pour les sorts d'héritage Tieffelin et les cantrips de lignage Elfe/Gnome).

**Critère d'utilité** : à la fin du plan, créer un Drakéide via le wizard force le joueur à choisir un type de dragon ; le souffle dans le mode Combat affiche le bon type de dégâts et la résistance correspondante ; le sort `breathWeaponDamageType` est utilisable par le moteur de dés. Un Tieffelin Infernal a `Fire Bolt` dans son mode Magie avec la caractéristique d'incantation choisie. Un Elfe Drow voit `Dancing Lights` + `Faerie Fire` (L3) + `Darkness` (L5) inscrits dans `knownSpells` / sorts d'héritage. Aucune ascendance avec sous-choix n'est créable sans poser ces choix.
