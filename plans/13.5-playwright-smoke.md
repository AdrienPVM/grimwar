## Goal

Câbler Playwright (déjà en `devDependencies`, le script `pnpm test:e2e` est déjà là — il manque la config et les tests) pour qu'un smoke headless attrape automatiquement la classe de bug « rien ne s'affiche / app ne se rend pas » qu'on vient de vivre. **Critère central** : si un futur plan rebrique la route `/` en placeholder, le smoke doit **casser**. Le smoke parcourt le chemin critique réel : `/` rend la LibraryScreen → CTA Créer → `/create` rend le formulaire → soumettre un perso minimal → atterrit sur `/character/:id` → la fiche rend (hero card + status strip + 5 mode tabs présents) → retour à la library via le nav shell → la liste contient le nouveau perso. En parallèle, ce plan **purge la dette e2e S1 accumulée** (plans 05, 06, 07, 08, 09, 10, 12, 12.5) : chaque ligne de la table consolidée devient soit un test, soit une requalification explicite.
